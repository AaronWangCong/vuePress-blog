## 它是如何使用查看的？

::: warning
事实上，一个好的博客网站在于标准化的分类，技术博文的规范以及查阅人的观感共同形成的，这也是我选用VuePress搭建此网站的初衷。搜索框可输入关键字或文章标题、分类标题、内容标题等查询。
